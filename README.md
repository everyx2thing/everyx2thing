## About

Computer Science student at Pusan National University working at the intersection of machine learning and real-time systems — from on-device personalization models to embedded RTOS programming. Currently building a context-aware AI safety framework for immersive XR as my graduation project.

`Unity` `C#` `PyTorch` `FastAPI` `PostgreSQL`
